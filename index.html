① YouTube에서 스크립트 복사하는 방법
영상 페이지 → 영상 하단 「 ··· 」더보기 클릭 → 「스크립트 열기」 클릭 → 텍스트 전체 선택 후 복사 (Ctrl+A → Ctrl+C)

② 영상 제목 입력 (선택)
제목을 입력하지 않으면 URL에서 자동으로 ID가 사용됩니다.
